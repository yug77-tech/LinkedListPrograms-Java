# LinkedListPrograms

Simple Java programs demonstrating linked list operations.

## Project structure

- [src](src): Java source files (`.java`)
- [bin](bin): Compiled class files (`.class`)

## Included programs

1. `Q1SinglyCreateDeleteDisplay` – Create, delete, and display in a singly linked list - Write a program that uses functions to perform the following:
a) Create a singly linked list of integers.
b) Delete a given integer from the above linked list.
c) Display (Traverse) the contents of the above list after deletion.

2. `Q2DoublyCreateDeleteDisplay` – Create, delete, and display in a doubly linked list - Write a program that uses functions to perform the following:
a) Create a doubly linked list of integers.
b) Delete a given integer from the above doubly linked list.
c) Display (Traverse) the contents of the above list after deletion.

3. `Q3InsertEndSingly` – Insert at end in a singly linked list - Write a program to insert a node at the end of a singly linked list and display all elements.

4. `Q4DeleteFirstSingly` – Delete first node in a singly linked list - Write a program to delete the first node of a singly linked list and display all elements.

5. `Q5SearchSinglyPosition` – Search node and show position in a singly linked list - Write a program to search for an element in a singly linked list and display its position.

6. `Q6InsertEndDoublyForward` – Insert at end in a doubly linked list (forward traversal) - Write a program to insert a node at the end in a doubly linked list and display the list forward.

7. `Q7DeleteFirstDoublyForward` – Delete first node in a doubly linked list (forward traversal) - Write a program to delete the first node in a doubly linked list and display the list forward.

8. `Q8TraverseDoublyBothDirections` – Traverse a doubly linked list in both directions - Write a program to traverse a doubly linked list in forward and backward direction.

9. `Q9DeleteNodeCircularDisplay` – Delete node and display in a circular linked list - Write a program to delete a node in a circular linked list and display all of its elements.

10. `Q10Insert25After20Circular` – Insert `25` after `20` in a circular linked list - Given circular list: 10 → 20 → 30 → 10, insert 25 after 20 and display traversal.

## Compile

From the project root:

```bash
javac -d bin src/*.java
```

## Run

Run any program by class name, for example:

```bash
java -cp bin Q3InsertEndSingly
```

You can replace `Q3InsertEndSingly` with any other class in this project.
