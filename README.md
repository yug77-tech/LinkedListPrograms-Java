# LinkedListPrograms

Simple Java programs demonstrating linked list operations.

## Project structure

- [src](src): Java source files (`.java`)
- [bin](bin): Compiled class files (`.class`)

## Included programs

1. `Q1SinglyCreateDeleteDisplay` – Create, delete, and display in a singly linked list
2. `Q2DoublyCreateDeleteDisplay` – Create, delete, and display in a doubly linked list
3. `Q3InsertEndSingly` – Insert at end in a singly linked list
4. `Q4DeleteFirstSingly` – Delete first node in a singly linked list
5. `Q5SearchSinglyPosition` – Search node and show position in a singly linked list
6. `Q6InsertEndDoublyForward` – Insert at end in a doubly linked list (forward traversal)
7. `Q7DeleteFirstDoublyForward` – Delete first node in a doubly linked list (forward traversal)
8. `Q8TraverseDoublyBothDirections` – Traverse a doubly linked list in both directions
9. `Q9DeleteNodeCircularDisplay` – Delete node and display in a circular linked list
10. `Q10Insert25After20Circular` – Insert `25` after `20` in a circular linked list

## Compile

From the project root:

```bash
javac -d bin src/*.java
```

## Run

Run any program by class name, for example:

```bash
java -cp bin Q3InsertEndSingly
```

You can replace `Q3InsertEndSingly` with any other class in this project.
